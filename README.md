## Robustez

- Single file, sin npm, sin build — funciona en `file://` y cualquier hosting
- IIFE pattern, sin ES modules
- Cada init en `safe()` try/catch
- Splash doble safety (CSS 4.5s + JS)

- `prefers-reduced-motion` respetado (parallax y mesh se desactivan)
- 100% responsive, mobile-first
- 44/44 checks pre-deploy · 0 errores JS · 40/40 reveals
